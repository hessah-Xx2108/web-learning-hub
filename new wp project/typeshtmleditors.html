Types of HTML Editors

Not sure what to use for coding? Here are some popular HTML editors - from simple to advanced!

Simple Text Editors

Good for beginners, but no special features. Just plain text.

Notepad (Windows)

Comes with Windows! Super basic but works. You type code, save as .html, open in browser.
✓ Simple ✗ No syntax highlighting
FREE

TextEdit (Mac)

Comes with Mac. Make sure to use plain text mode (Format > Make Plain Text).
✓ Simple ✗ Have to change settings
FREE

Code Editors (Most Popular)

Made specifically for coding. Syntax highlighting, autocomplete, etc.

VS Code

The most popular editor right now! Made by Microsoft. Has extensions for everything.
✓ Extensions ✓ Git integration
Download FREE

Sublime Text

Super fast and lightweight. Many people love it. Has a forever free trial.
✓ Fast ✗ Nags you to buy
Download FREE trial

Atom

Made by GitHub. Hackable to the core. Good for beginners.
✓ Customizable ✗ Can be slow
Download FREE

Brackets

Made for web design. Has live preview - see changes in real time!
✓ Live preview ✗ Not updated much
Download FREE

Notepad++

Like Notepad but better. Windows only. Lightweight and fast.
✓ Lightweight ✗ Windows only
Download FREE

Online Editors (No install needed!)

Just open your browser and start coding. Good for testing small things.

CodePen

See HTML, CSS, and JavaScript results instantly. Great for sharing code.
Open FREE

JSFiddle

Test your HTML, CSS, JS online. Can include libraries easily.
Open FREE

Replit

Full development environment in browser. Supports many languages.
Open FREE

W3Schools TryIt

Great for learning. Edit examples and see results immediately.
Open FREE

IDEs (For professionals)

Big, powerful, but might be too much for beginners.

WebStorm

Made by JetBrains. Very smart autocomplete. Great for JavaScript.
PAID

Dreamweaver

Old but still used. Has visual editor (see design as you code).
PAID

Tips for beginners:

  • Start with VS Code - it's free, popular, and has everything you need
  • Learn some keyboard shortcuts - they save so much time!
  • Try online editors first if you don't want to install anything
  • Use extensions! For web dev: Live Server, Prettier, ESLint
  • Don't start with paid IDEs until you know what you're doing

What do we use?

Most of us use VS Code with these extensions:

  • Live Server - to see changes instantly
  • Prettier - formats code automatically
  • Bracket Pair Colorizer - makes brackets easier to see
  • HTML CSS Support - autocomplete for classes
