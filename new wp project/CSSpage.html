CSS

CSS (Cascading Style Sheets) is what makes websites look nice! It controls colors, fonts, layouts, and even animations.

Without CSS, websites would just be black text on white background. Boring! CSS adds the style.

Example:
h1 {
  color: blue;
  font-size: 24px;
}

You can write CSS in the same file or in separate .css files. External stylesheets are best for multiple pages.
