# Web Development Learning Hub

An educational website designed to help beginners learn web programming fundamentals through interactive content and practical examples.

## Overview

Web Development Learning Hub is a 10+ page educational platform that introduces core web technologies (HTML, CSS, JavaScript) to aspiring developers. The site features responsive navigation, interactive content, and user authentication forms to demonstrate real-world web development concepts.

## Features

- **Multi-Page Structure:** 10+ interconnected pages covering various web development topics
- **Responsive Navigation:** Dropdown menus for easy access to different sections
- **Interactive Content:** "Read More" toggle functionality for better content management
- **User Authentication:** Registration and login forms with validation
- **Clean Design:** Educational-focused UI with clear typography and layout

## Built With

- **HTML5** - Semantic markup for content structure
- **CSS3** - Styling, layout, and responsive design
- **JavaScript** - Interactive features and form validation

## Topics Covered

The website includes pages on:
- HTML basics and semantic elements
- CSS styling and layout techniques
- JavaScript fundamentals
- Form handling and validation
- Responsive web design principles
- Navigation and information architecture
